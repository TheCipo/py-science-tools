# Python Mini Tools

A collection of small Python scripts for practical and scientific calculations.

## Scripts

- **birthday_paradox.py** – calculates the probability of shared birthdays in a group.  
- **density_altitude.py** – computes density altitude from temperature and pressure, with airtow guidance.  
- **microwave_cooking_time.py** – estimates microwave cooking time based on food mass, temperature, and power.  
- **saltedwater.py** – calculates boiling point of water with added salt.
